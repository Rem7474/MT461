# MT461 - Travaux Pratiques

Ce dépôt contient les travaux pratiques du cours MT461, organisés en plusieurs TPs avec différents exercices en R.

## Structure du projet

```
MT461/
├── TP1/
│   ├── Partie 1/
│   │   ├── Ex1.R
│   │   ├── Ex2.R
│   │   └── output/
│   └── Partie 2/
│       ├── Ex3.R
│       ├── Ex4.R
│       ├── Ex5.R
│       └── output/
└── TP2/
    ├── Partie 1/
    │   ├── Ex1.R
    │   └── output/
    └── Partie 2/
        ├── Ex2.R
        ├── Ex3.R
        └── output/
```

## Description

### TP1
- **Partie 1** : Exercices 1 et 2
- **Partie 2** : Exercices 3, 4 et 5

### TP2
- **Partie 1** : Exercice 1
- **Partie 2** : Exercices 2 et 3

## Utilisation

### Prérequis
- R (version 4.0 ou supérieure recommandée)
- RStudio (optionnel mais recommandé)

### Exécution des scripts
1. Ouvrir le script R souhaité dans RStudio ou votre éditeur préféré
2. Exécuter le script ligne par ligne ou en entier
3. Les sorties générées seront stockées dans les dossiers `output/` correspondants

### Organisation des sorties
Chaque partie dispose d'un dossier `output/` dédié pour stocker :
- Les graphiques générés
- Les résultats d'analyse
- Les fichiers de données exportés

## Notes
- Assurez-vous d'avoir les packages R nécessaires installés avant d'exécuter les scripts
- Les chemins relatifs sont utilisés pour faciliter la portabilité du projet
